# Conways' Game of Life
## rust-wasm preview
https://eval-exec.github.io/conways-game-of-life/
## run
```bash
cargo install wasm-pack
wasm-pack build --target web

python -m http.server

# then open your web browser and visit http://127.0.0.1:8000
```
